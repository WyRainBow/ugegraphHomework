# Apache HugeGraph 毕业项目实现计划

## 项目目标

在 `/Users/wy770/Apache` 目录下创建一个完整的 Python 项目，实现两个核心任务：

1. **毕业文章生成**：生成 HugeGraph 从 Apache 孵化器毕业的公众号风格文章，包含正文、数据可视化图表和封面图
2. **投票结果统计**：从 Apache 邮件列表提取投票数据，按分组维度统计并输出结果

## 设计理念

本项目采用模块化、工作流驱动的设计模式，根据实际需求进行了设计：

- **设计模式**：Workflow 分阶段执行、统一的 LLM 调用接口、Prompt 模板管理
- **实现方式**：直接使用函数调用，简化实现
- **使用已有数据**：直接使用提供的投票统计和需求分析文档，无需重新提取
- **专注核心功能**：实现文章生成和投票统计两个核心任务

**重要说明**：

- 采用模块化设计，便于维护和扩展
- 根据实际需求设计，满足项目功能要求
