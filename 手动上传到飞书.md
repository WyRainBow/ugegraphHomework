# 手动上传内容到飞书文档指南

由于自动写入内容失败，你可以手动将内容复制到飞书文档中。

   - 链接: https://bytedance.feishu.cn/docx/R6cGdejzaofVw7xKEVNchnNEnge
   - 源文件: `docs/实现思路和Prompt说明.md`

2. **文章生成 Prompt**
   - 文档 ID: `H6SjdziUqoOrMvxVjEFc0rLpnec`
   - 链接: https://bytedance.feishu.cn/docx/H6SjdziUqoOrMvxVjEFc0rLpnec

3. **大纲生成 Prompt**
   - 文档 ID: `B7QddXT8yooiGrxSdpocV0yfnXb`
   - 链接: https://bytedance.feishu.cn/docx/B7QddXT8yooiGrxSdpocV0yfnXb

   - 源文件: `config/prompts/quote_extraction.md`

## 操作步骤

1. 打开对应的飞书文档链接
2. 打开本地对应的 Markdown 源文件
3. 复制 Markdown 文件内容
4. 在飞书文档中粘贴（飞书会自动识别 Markdown 格式）

## 或者使用命令行快速打开文件

在终端运行：

```bash
# 打开实现思路文档
open docs/实现思路和Prompt说明.md

# 打开文章生成 Prompt
open config/prompts/article_generation.md

# 打开大纲生成 Prompt
open config/prompts/outline_generation.md

# 打开引用提取 Prompt
open config/prompts/quote_extraction.md
```

然后复制内容到对应的飞书文档中。
