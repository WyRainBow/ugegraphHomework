# 里程碑！Apache HugeGraph 成功毕业，正式成为顶级项目！

![Cover Image](../images/hugegraph_cover_image.png)

我们激动地宣布，Apache HugeGraph 项目已成功从 Apache 孵化器毕业，正式晋升为 Apache 顶级项目（TLP）！这标志着 HugeGraph 在技术成熟度、社区活跃度和项目治理方面均达到了 Apache 软件基金会的最高标准。

## 项目简介：Apache HugeGraph 是什么？

Apache HugeGraph 是一个开源的分布式图数据库，专为海量数据存储和复杂关系查询设计。它提供了强大的图计算能力和灵活的数据模型，广泛应用于金融风控、电商推荐、电信诈骗识别等多个领域，帮助用户深入挖掘数据中的关联价值。

## 孵化历程回顾：四年磨砺，破茧成蝶

自2022年1月进入 Apache 孵化器以来，HugeGraph 经历了四年的持续发展与蜕变。在这段旅程中，项目团队与全球社区成员紧密协作，不断完善产品功能，优化性能，并积极拥抱 Apache 之道，逐步构建起一个健康、开放、多元的社区。

## 关键成就亮点

### 社区成长：贡献者与领导者并进

孵化期间，HugeGraph 社区蓬勃发展，吸引了超过210位贡献者加入。项目成功培养了10位新 Committer 和2位新 PPMC 成员，为社区注入了新鲜血液与领导力量，共同推动项目向前发展。

![Community Trend](../images/community_trend.png)

### 版本发布：稳定迭代，功能创新

在孵化期间，HugeGraph 稳定发布了5个 Apache 版本，从 1.0 到 1.7。这得益于5位发布经理的辛勤付出和严格把关，确保了每个版本的质量与稳定性，持续为用户带来新功能和性能提升。

![Release Timeline](../images/release_timeline.png)

### 生产应用：行业深耕，价值凸显

HugeGraph 已在多个关键行业实现规模化生产应用，包括金融风控、电商推荐、电信诈骗识别等。其卓越的性能和稳定性，帮助企业解决了海量复杂关系数据的存储与分析挑战，创造了显著的业务价值。

### 生态系统：开放融合，共创未来

项目积极融入 Apache 生态，与 Apache SeaTunnel、Apache TinkerPop 等项目实现深度集成。同时，HugeGraph 积极参与 Google Summer of Code (GSoC) 和开源软件供应链点亮计划 (OSPP)，吸引更多学生和开发者参与开源贡献。

### 治理合规：遵循 Apache 之道

在孵化过程中，HugeGraph 严格遵循 Apache 软件基金会的各项政策和流程。PMC 成员的确认和项目治理的规范化，确保了项目以开放、透明、协作的方式健康发展，为毕业奠定了坚实基础。
